Tentang Saya

Mahasiswa Teknik Informatika yang memiliki keahlian dalam pengembangan web menggunakan HTML, CSS, JavaScript, dan PHP. Dengan pengalaman praktis dalam membangun aplikasi dan situs web, saya familiar dengan berbagai framework seperti Bootstrap, Tailwind, dan Laravel. Selain itu, saya juga memiliki kemampuan dalam pengelolaan database menggunakan MySQL dan SQLite. Dengan kemampuan analisis yang kuat, orientasi pada detail, dan komitmen untuk terus belajar, saya siap berkontribusi dalam menciptakan solusi teknologi yang inovatif dan bermanfaat bagi masyarakat luas. Saya antusias untuk bekerja dalam tim yang dinamis dan menghadapi tantangan di bidang teknologi informasi.
